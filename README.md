
# Mini Quiz App

website to answer funny question

## Backend

To deploy this project run

```bash
  cd backend
```
```bash
  npm install
```
```bash
  npx prisma generate
```
```bash
  npx prisma db push
```bash
  npx prisma db seed
```bash
  npm run dev
```

## Frontend

To deploy this project run

```bash
  cd frontend
```
```bash
  npm install
```
```bash
  npm run dev
```
